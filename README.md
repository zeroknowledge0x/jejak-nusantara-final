# Jejak Nusantara Final

Game Roblox edukasi budaya Indonesia - Versi Final siap kumpul.

## Team
- Project Manager: Daffa Rifqi A.F
- Team: Mahda Vidho Pratama, Sakti Hermawan, Muhammad Rakha Q.A

## Features
- 4 Level: Desa Budaya, Sanggar Seni, Pasar Tradisional, Tempat Bersejarah
- Mini Games: Angklung, Tari, Tawar-menawar, Puzzle Sejarah
- Narrative Adventure + Education
- Cultural Insight System
- Multiple Endings

## Files
- `GameServer.lua` — Server-side logic, data management, game state
- `GameClient.lua` — Client-side UI, interactions, mini-games
- `LevelDesign.lua` — Level layouts, NPC placement, objectives
